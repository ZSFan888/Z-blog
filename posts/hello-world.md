# 博客上线了！

经过一番折腾，博客终于正式上线了 🎉

这个博客基于 **React + Vite** 构建，部署在 **Cloudflare Pages** 上，全球 CDN 加速，访问极速。

## 技术栈

- **前端框架**：React 18 + TypeScript
- **构建工具**：Vite 5
- **样式方案**：Tailwind CSS
- **动画**：Framer Motion
- **部署平台**：Cloudflare Pages
- **内容管理**：自研后台 + GitHub API

## 为什么选择 Cloudflare Pages？

Cloudflare Pages 有几个让我很满意的特点：

1. **免费额度慷慨**：每月 500 次构建，带宽无限制
2. **全球 CDN**：300+ 个节点，国内访问速度也不错
3. **自动 HTTPS**：绑定域名后自动签发证书
4. **Git 集成**：推送代码自动触发构建，CI/CD 开箱即用
